fui Find unused Objective-C imports.

Unused A Mac app for checking Xcode projects for unused resources

https://github.com/kyleduo/TinyPNG4Mac

RevealApp UI 可视化调试工具 http://revealapp.com/
Watchdog Xcode 缓存自动清理工具 https://www.cerebralgardens.com/watchdog
Hopper Disassembler The OS X and Linux Disassembler

Paw The most advanced HTTP client for Mac

Dash API 文档查看和代码管理工具，好用到哭 https://kapeli.com/dash
xScope A powerful set of tools that are ideal for measuring, inspecting & testing on-screen graphics and layouts.
